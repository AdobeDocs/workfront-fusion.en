## Make a custom API call (Polling) module now available in Adobe Authenticator connector

To make it possible to repeat API calls until a certain condition or limit is met, we've created the Make a custom API call (Polling) module. This module includes options that allow you to configure repeat API calls. You can have the API call execute until a particular condition is met, set a specific number of retries, and configure the time between retries.

For more information, see [Make a custom API call (Polling)](/help/workfront-fusion/references/apps-and-modules/adobe-connectors/adobe-authenticator-modules.md#make-a-custom-api-call-polling) in the article Adobe Authenticator modules.
