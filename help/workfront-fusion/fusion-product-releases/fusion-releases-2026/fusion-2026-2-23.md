# Workfront Fusion release activity: Week of February 23, 2026

This page describes all enhancements made in Adobe Workfront Fusion the week of February 23, 2026.

For a list of all recent changes, see [Adobe Workfront Fusion release activity](/help/workfront-fusion/fusion-product-releases/fusion-release-activity.md).

For a list of recent bug fixes in Workfront Fusion, see the [Workfront Maintenance Updates](https://experienceleague.adobe.com/en/docs/workfront-known-issues/releases/current-updates) page and check for any updates labeled Workfront Fusion Maintenance Update.

New Veeva Vault modules

To expand the capabilities of the Veeva Vault connector, we've added the following modules:

* Create a single document relationship
* Create multiple document relationships
* Create multiple annotations
* Get document annotations
* Get document relationships
* Update multiple annotations
* Delete a single document relationship
* Delete multiple document relationships
* Delete multiple annotations

For more information, including instructions, see [Veeva Vault modules](/help/workfront-fusion/references/apps-and-modules/third-party-connectors/veeva-vault-modules.md).
