# Workfront Fusion release activity: Week of May 25, 2026

This page describes all enhancements made in Adobe Workfront Fusion the week of May 25, 2026.

For a list of all recent changes, see [Adobe Workfront Fusion release activity](/help/workfront-fusion/fusion-product-releases/fusion-release-activity.md).

For a list of recent bug fixes in Workfront Fusion, see the [Workfront Maintenance Updates](https://experienceleague.adobe.com/en/docs/workfront-known-issues/releases/current-updates) page and check for any updates labeled Workfront Fusion Maintenance Update.

## Webhook priority updates restricted to specific roles

Updating a webhook's priority is now restricted to users with the [!UICONTROL Team Admin] role, or with the organization [!UICONTROL Admin] or [!UICONTROL Owner] role. For other users, the Priority dropdown in the Webhooks area is disabled.

For more information, see [Instant triggers (webhooks)](/help/workfront-fusion/references/modules/webhooks-reference.md).
