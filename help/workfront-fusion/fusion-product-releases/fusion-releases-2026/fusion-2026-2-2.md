# Workfront Fusion release activity: Week of February 2, 2026

This page describes all enhancements made in Adobe Workfront Fusion the week of February 2, 2026.

For a list of all recent changes, see [Adobe Workfront Fusion release activity](/help/workfront-fusion/fusion-product-releases/fusion-release-activity.md).

For a list of recent bug fixes in Workfront Fusion, see the [Workfront Maintenance Updates](https://experienceleague.adobe.com/en/docs/workfront-known-issues/releases/current-updates) page and check for any updates labeled Workfront Fusion Maintenance Update.

## Updates to the Scenarios area

To make scenarios easier to manage, we've some changes to the Scenarios area. Now you can:

* View and edit worker pool assignments 
* Move scenarios to folders in bulk

For more information on using the Scenarios area to manage scenarios, see [Manage scenarios from the Scenarios area](/help/workfront-fusion/manage-scenarios/manage-scenarios-from-list.md).
