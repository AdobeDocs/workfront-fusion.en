# Workfront Fusion release activity: Week of March 30, 2026

This page describes all enhancements made in Adobe Workfront Fusion the week of March 30, 2026.

For a list of all recent changes, see [Adobe Workfront Fusion release activity](/help/workfront-fusion/fusion-product-releases/fusion-release-activity.md).

For a list of recent bug fixes in Workfront Fusion, see the [Workfront Maintenance Updates](https://experienceleague.adobe.com/en/docs/workfront-known-issues/releases/current-updates) page and check for any updates labeled Workfront Fusion Maintenance Update.

## New table style in Data stores and Data structures areas

To create a cleaner and more consistent experience, we've updated the Data stores and Data structures area with a new table style. The tables now match the styling and functionality of other areas of Fusion.

For more information, see:

* [Data stores](/help/workfront-fusion/create-scenarios/map-data/data-stores.md)
* [Data structures](/help/workfront-fusion/references/mapping-panel/data-types/data-structures.md)
