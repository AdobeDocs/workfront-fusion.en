# Workfront Fusion release activity: Week of June 15 2026

This page describes all enhancements made in Adobe Workfront Fusion the week of June 15 2026.

For a list of all recent changes, see [Adobe Workfront Fusion release activity](/help/workfront-fusion/fusion-product-releases/fusion-release-activity.md).

For a list of recent bug fixes in Workfront Fusion, see the [Workfront Maintenance Updates](https://experienceleague.adobe.com/en/docs/workfront-known-issues/releases/current-updates) page and check for any updates labeled Workfront Fusion Maintenance Update.

## Upcoming deprecation of legacy Photoshop modules

Adobe Photoshop has deprecated some elements of its API, which Fusion uses to perform actions in Photoshop. 

**Therefore, some of the existing Photoshop modules will not work after July 30, 2026.**

We recommend updating any scenarios that use these modules to the updated modules as soon as possible. 

For a list of affected modules, see [Adobe Photoshop API deprecation updates](/help/workfront-fusion/references/apps-and-modules/adobe-connectors/adobe-photoshop-modules.md#adobe-photoshop-api-deprecation-updates).

For an explanation of how API changes affect Workfront Fusion, see [Overview of APIs in Fusion](/help/workfront-fusion/get-started-with-fusion/understand-fusion/api-overview.md).


## New endpoints now available for Adobe Workfront Fusion API

To make the Adobe Workfront Fusion API more flexible and powerful, we've added the endpoint for the following actions:

* **Scenarios**: List scenarios, get scenarios by ID, clone a scenario, export blueprints, and get scenario dependencies
* **Executions**: List execution and get a single execution
* **Connections**: List connections, get a connection by ID, and list scenarios using a connection
* **Data stores**: List data stores by team and get a data store by ID
* **Folders**: List, create, update, and delete folders
* **Webhooks**: List webhooks and get a webhook by ID

To view the complete changelog, see [Changelog](https://developer.adobe.com/workfront-fusion-apis/getting-started/changelog/) in the Workfront Fusion developer documentation.

## Adobe Workfront Unified Review and Approvals connector now available

To expand Fusion's ability to working with unified review and approvals in Workfront, we've created the Adobe Workfront Unified Review and Approvals connector. With the Adobe Workfront Unified Review and Approvals:

* Create, update, or delete approvals and stages
* Add or remind approval participants
* Make or delete decisions
* Retrieve information about approvals, templates, or approval bots
* Get suggested approvals or participants
* Make a custom API request

For information and instructions, see [Adobe Workfront Unified Review and Approvals modules](/help/workfront-fusion/references/apps-and-modules/adobe-connectors/workfront-review-and-approvals-modules.md).

## New modules for Adobe Photoshop connector

To expand the capabilities of the Adobe Photoshop connector, we've added these modules:

* Create an artboard
* Create or edit a composite
* Edit an image with various adjustments
* Execute Photoshop actions, scripts, and transformations
* Generate a manifest

For information and instructions for the new modules, see [Adobe Photoshop modules](/help/workfront-fusion/references/apps-and-modules/adobe-connectors/adobe-photoshop-modules.md).

## New modules available for Adobe InDesign connector

To expand the capabilities of the Adobe InDesign connector, we've added these modules:

* Submit a custom script
* Update custom script app version
* Get current app versions
* Convert PDF to InDesign document

For information and instructions for the new modules, see [InDesign modules](/help/workfront-fusion/references/apps-and-modules/adobe-connectors/indesign-modules.md).
