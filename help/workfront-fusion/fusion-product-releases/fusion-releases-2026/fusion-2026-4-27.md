# Workfront Fusion release activity: Week of April 27, 2026

This page describes all enhancements made in Adobe Workfront Fusion the week of April 27, 2026.

For a list of all recent changes, see [Adobe Workfront Fusion release activity](/help/workfront-fusion/fusion-product-releases/fusion-release-activity.md).

For a list of recent bug fixes in Workfront Fusion, see the [Workfront Maintenance Updates](https://experienceleague.adobe.com/en/docs/workfront-known-issues/releases/current-updates) page and check for any updates labeled Workfront Fusion Maintenance Update.

## New modules available for Veeva Vault connector

To expand the capabilities of the Veeva Vault connector, we've added the following new modules:

* Extract multiple files
* Retrieve extract results
* Load multiple files
* Retrieve load results
* List items at a path

For information about Veeva Vault modules, including the new modules, see [Veeva Vault modules](/help/workfront-fusion/references/apps-and-modules/third-party-connectors/veeva-vault-modules.md).
