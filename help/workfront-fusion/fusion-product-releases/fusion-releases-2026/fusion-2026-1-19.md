# Workfront Fusion release activity: Week of January 19, 2026

This page describes all enhancements made in Adobe Workfront Fusion the week of January 19, 2026.

For a list of all recent changes, see [Adobe Workfront Fusion release activity](/help/workfront-fusion/fusion-product-releases/fusion-release-activity.md).

For a list of recent bug fixes in Workfront Fusion, see the [Workfront Maintenance Updates](https://experienceleague.adobe.com/en/docs/workfront-known-issues/releases/current-updates) page and check for any updates labeled Workfront Fusion Maintenance Update.

## Update keys

To make it easier for you to use keys in your Fusion scenarios, we've added the ability to update existing keys. Now, you can update the key directly. Any scenarios that use the key will use the updated version. 

For more information on keys, see [Keys](/help/workfront-fusion/references/modules/keys.md).
