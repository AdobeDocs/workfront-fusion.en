# Workfront Fusion release activity: Week of November 3, 2025

This page describes all enhancements made in Adobe Workfront Fusion the week of November 3, 2025.

For a list of all recent changes, see [Adobe Workfront Fusion release activity](/help/workfront-fusion/fusion-product-releases/fusion-release-activity.md).

For a list of recent bug fixes in Workfront Fusion, see the [Workfront Maintenance Updates](https://experienceleague.adobe.com/en/docs/workfront-known-issues/releases/current-updates) page and check for any updates labeled Workfront Fusion Maintenance Update.

## New and updated modules now available for Frame.io connector

To allow you to interact directly with the metadata on Frame.io assets, we've added several new modules to the Frame.io connector. With the new modules, you can: 

* Create, update, or delete account-level metadata descriptions
* Get a list of available metadata fields
* Retrieve or update metadata field values

To expand the capability of the Create an asset module, we've also made an update there. Now, you can choose whether to upload local files or remote files. 

To expand the capability of the Create an asset module, For information on the Frame.io connector, see [Frame.io modules](/help/workfront-fusion/references/apps-and-modules/adobe-connectors/frame-io-modules-new.md).
